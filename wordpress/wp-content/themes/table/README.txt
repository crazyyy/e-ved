Table WordPress Theme 1.0
http://www.theme-junkie.com/themes/table

INSTALL: 
1. Upload the theme folder /table via FTP to your wp-content/themes/ directory.
2. Go to your WordPress dashboard and activate the Table theme.
 Inside your WordPress dashboard, go to Appearance > Table Theme Options and configure them to your liking.

If you are looking for theme support, please visit http://www.theme-junkie.com/forum/
